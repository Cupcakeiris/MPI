Решить уравнение переноса численно и сравнить результаты с аналитическим решением, используя два метода аппроксимации производной по пространству.

 

1. Методы аппроксимации:
    - Центральная разностная схема по пространству
    - Схема производной назад по пространству



2. Используйте аналитическое решение уравнения для проверки корректности численного метода.
3. Требования к программе:
    - Реализуйте коды для обоих методов аппроксимации производной и сравните их с аналитическим решением.

    - Используйте MPI для параллельной обработки численного решения.
    - Для коммуникации между процессами используйте те MPI-функции, которые были рассмотрены на лекциях
    - Постройте графики численных решений и аналитического решения, сравнив погрешности между ними.
    - Используйте явные численные методы решения уравнения

 

Ожидаемые результаты:
1. Код программы, который корректно решает уравнение переноса параллельно.
2. Графики численного и аналитического решений.
3. Анализ полученных данных, включая погрешности и сравнение времени выполнения для разных методов.
