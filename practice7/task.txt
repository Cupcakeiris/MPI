2. Разработать программу для численного решения уравнения Пуассона с использованием метода конечных разностей.

3. Реализовать параллельное решение задачи с использованием MPI, распараллелив вычисления для различных частей интервала.

4. Сравнить результаты аналитического и численного решений.

5. Построить графики для сравнения результатов аналитического и численного решений с использованием TechPlot, Python, Excel или др.

6. Вычислите численную ошибку 

 

Требования:

- Использовать только те MPI-функции, которые были пройдены в курсе.

- Добавьте тесты для проверки точности решения при различном количестве делений N.
- Оформите код с комментариями, объясняющими ключевые шаги.
- Необходимо провести анализ времени вычисления для различных чисел процессов.
